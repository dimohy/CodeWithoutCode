# 생각을 코드로

> AI와 함께하는 컴퓨팅 사고 - 일반인편

**바이브 코딩을 위한 컴퓨팅 사고 시리즈 1권**

---

## 이 책에 대하여

AI와 함께하는 새로운 프로그래밍 패러다임, **바이브 코딩**을 배우는 15주차 강의 시리즈입니다.

바이브 코딩은 코딩을 직접 작성하지 않고도 문제를 해결할 수 있는 새로운 방식입니다. GitHub Copilot과 함께 사고하고 협업하는 방법을 배우며, 컴퓨팅 사고의 4대 원리(분해, 패턴 인식, 추상화, 알고리즘적 사고)를 체계적으로 학습합니다.

---

## 대상 독자

프로그래밍 경험이 없거나 적은 일반인을 위한 책입니다.

코딩 경험이 없어도 괜찮습니다. 이 책은 여러분이 이미 일상에서 사용하고 있는 문제 해결 능력을 컴퓨팅 사고로 발전시켜 드립니다. 전문 용어 대신 쉬운 일상 언어로 설명하며, GitHub Copilot이라는 친구와 함께 자연스럽게 문제를 해결하는 방법을 배웁니다.

---

## 학습 목표

- 컴퓨팅 사고의 4대 원리 이해 및 적용
- GitHub Copilot과의 효과적인 대화 방법 습득
- 실생활 문제를 바이브 코딩으로 해결하는 능력 배양
- GitHub Copilot Agent 모드 중급 수준 활용

---

## 주요 특징

- **코드를 읽지 않아도 되는 학습 방식**: 결과와 과정에 집중합니다
- **일상 속 사례로 쉽게 이해**: 복잡한 개념을 친숙한 예시로 설명합니다
- **따뜻하고 공감적인 문체**: 함께 배워가는 느낌으로 작성되었습니다
- **단계별 실습과 프롬프트 작성 연습**: 직접 해보며 배웁니다

- **2권: 사고의 깊이** - 전문가를 위한 바이브 코딩과 컴퓨팅 사고
