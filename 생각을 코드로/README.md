# 생각을 코드로

> 누구나 할 수 있는 바이브 코딩 사고

**바이브 코딩을 위한 컴퓨팅 사고 시리즈 1권**

---

## 환영합니다

코딩 경험이 없어도 괜찮습니다. 이 책은 여러분이 이미 일상에서 사용하고 있는 문제 해결 능력을 컴퓨팅 사고로 발전시켜 드립니다.

**바이브 코딩(Vibe Coding)**은 코딩을 직접 작성하지 않고도 문제를 해결할 수 있는 새로운 프로그래밍 패러다임입니다. GitHub Copilot이라는 친구와 함께 자연스럽게 문제를 해결하는 방법을 배우며, 컴퓨팅 사고의 4대 원리(분해, 패턴 인식, 추상화, 알고리즘적 사고)를 체계적으로 학습합니다.

전문 용어 대신 쉬운 일상 언어로 설명하며, 코드를 읽지 않아도 되는 학습 방식으로 진행됩니다. 여러분은 **무엇을 요청할지** 생각하고 **어떤 결과를 얻었는지** 확인하는 데 집중하게 됩니다.

- 실생활 문제를 바이브 코딩으로 해결하는 능력 배양

## 이 책의 특징

### 코드를 읽지 않아도 되는 학습 방식

이 책은 코드를 읽거나 이해하는 것이 아니라, **무엇을 요청할지** 생각하고 **어떤 결과를 얻었는지** 확인하는 데 집중합니다. 결과와 과정 중심의 학습으로 누구나 바이브 코딩을 시작할 수 있습니다.

### 일상 속 사례로 쉽게 이해

복잡한 프로그래밍 개념을 일상생활의 친숙한 예시로 풀어냅니다. 전문 용어는 최소화하고, 필요시 괄호 안에 쉬운 설명을 추가합니다.

### 따뜻하고 공감적인 문체

독자에게 직접 말을 거는 듯한 친근한 톤으로 작성되었습니다. 함께 배워가는 느낌으로, 실패와 시행착오를 자연스러운 학습 과정으로 받아들이도록 독려합니다.

### GitHub Copilot Agent 모드 마스터하기

이 책의 핵심 도구는 **GitHub Copilot Agent 모드**입니다. 15주 과정을 마치면 GitHub Copilot Agent를 **중급 수준으로 활용**할 수 있게 됩니다.

- 효과적인 대화 방법과 프롬프트 작성법
- 문제를 단계적으로 해결하는 협업 패턴
- 실생활 문제를 프로그램으로 만드는 과정

## 학습 방법

### 이렇게 읽으세요

**코드를 보지 않아도 됩니다**. 이 책은 코드를 읽거나 이해하는 것이 아니라, **무엇을 요청할지** 생각하고 **어떤 결과를 얻었는지** 확인하는 데 집중합니다.

1. **순서대로 읽으세요**: 각 챕터는 이전 내용을 기반으로 구성되어 있습니다.
2. **실습을 반드시 따라하세요**: 읽기만 해서는 부족합니다. GitHub Copilot과 직접 대화하며 경험을 쌓으세요.
3. **완벽을 추구하지 마세요**: 처음부터 완벽한 프롬프트를 작성할 필요는 없습니다. 시행착오도 학습 과정의 일부입니다.
4. **여러분의 언어로 표현하세요**: 프로그래밍 용어를 몰라도 괜찮습니다. 일상 언어로 문제를 설명하는 연습을 하세요.

### 15주 학습 흐름

- **1-3주차**: 바이브 코딩과 컴퓨팅 사고 소개
- **4-5주차**: 컴퓨팅 사고 4대 원리 학습
- **6주차**: GitHub Copilot과의 협업 방법
- **7-9주차**: 실습 및 4대 원리 종합 연습
- **10-11주차**: 실생활 문제 해결
- **12-13주차**: 최종 프로젝트
- **14-15주차**: 평가 및 미래 전망

## 이 책을 마치면

15주 과정을 완료하면:

✅ 컴퓨팅 사고의 4대 원리를 이해하고 적용할 수 있습니다  
✅ GitHub Copilot과 효과적으로 대화하며 협업합니다  
✅ 실생활 문제를 바이브 코딩으로 해결할 수 있습니다  
✅ GitHub Copilot Agent 모드를 중급 수준으로 활용합니다  
✅ 프로그래밍의 기초를 이해하고 더 깊이 배울 준비가 됩니다
