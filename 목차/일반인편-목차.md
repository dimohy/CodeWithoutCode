# 《생각을 코드로: 누구나 할 수 있는 바이브 코딩 사고》

**부제**: AI와 함께하는 새로운 문제 해결의 언어

---

## 15주차 강의 목차 (바이브 코딩 중심 컴퓨팅 사고 교육)

### 1주차: 오리엔테이션 - 코딩 없는 코딩의 시대
- 바이브 코딩이란 무엇인가
- AI 시대의 문제 해결 방식
- 강의 개요 및 학습 목표

### 2주차: 컴퓨팅 사고의 필요성
- 왜 컴퓨팅 사고가 중요한가
- 일상 속 컴퓨팅 사고 사례
- 전통적 코딩 vs 바이브 코딩

### 3주차: 바이브 코딩의 이해와 AI 도구 탐색
- 바이브 코딩의 개념과 특징
- ChatGPT, Claude, Copilot 등 AI 도구 소개
- 실습: GitHub Copilot으로 첫 바이브 코딩 경험

### 4주차: 컴퓨팅 사고의 4대 원리 I - 분해와 패턴 인식
- 분해(Decomposition): 큰 문제를 작은 문제로
- 패턴 인식(Pattern Recognition): 반복되는 규칙 발견
- 일상 생활 속 사례로 이해하기
- GitHub Copilot에 문제를 설명하는 기초

### 5주차: 컴퓨팅 사고의 4대 원리 II - 추상화와 알고리즘
- 추상화(Abstraction): 핵심만 남기기
- 알고리즘적 사고(Algorithmic Thinking): 단계별 문제 해결
- 4대 원리의 통합적 적용
- GitHub Copilot에게 효과적으로 지시하기

### 6주차: GitHub Copilot과 협업하는 사고법
- GitHub Copilot Agent와의 효과적인 대화 방법
- 프롬프트 엔지니어링 기초
  - 명확한 목표 제시
  - 충분한 컨텍스트 제공
  - 단계별 개선 요청
- 반복적 개선 전략
- AI의 강점과 한계 이해하기

### 7주차: 실습 - 분해와 패턴 인식 연습
- 일상 문제를 작은 단계로 나누기
- 예제: 아침 준비 루틴, 가계부 데이터 패턴 분서
- GitHub Copilot과 함께 분해와 패턴 찾기
- 프롬프트 작성 패턴 연습

### 8주차: 실습 - 추상화와 알고리즘 훈련
- 핵심만 남기고 불필요한 디테일 제거하기
- 단계별 문제 해결 지시문 작성
- 예제: 교통 경로 단순화, 레시피를 알고리즘으로
- GitHub Copilot과 함께 4대 원리 통합 연습

### 9주차: 중간평가 + 4대 원리 종합 연습
- 중간평가 (1-8주차 내용)
- 4대 원리를 통합한 문제 해결
- 실제 문제에 4대 원리 적용 연습
- GitHub Copilot 활용 패턴 점검

### 10주차: 실습 - GitHub Copilot에게 문제 정의하고 해결하기
- 효과적인 프롬프트 작성 패턴
  - 문제 정의 방법
  - 예상 결과 명시
  - 제약 조건 설명
- GitHub Copilot 응답 평가 및 개선하기
- 실습: 다양한 유형의 문제를 GitHub Copilot과 함께 해결

### 11주차: 실습 - 바이브 코딩으로 실생활 문제 해결
- 개인 맞춤형 문제 선정 및 분석
- 4대 원리를 적용한 문제 분해
- GitHub Copilot과 협업하여 솔루션 설계
- 예제: 일정 관리, 데이터 정리, 콘텐츠 생성 등
- 협업 패턴 및 베스트 프랙티스

### 12주차: 바이브 코딩 프로젝트 I - 기획
- 최종 프로젝트 주제 선정
- 문제 정의 및 분석
- GitHub Copilot 활용 전략 수립

### 13주차: 바이브 코딩 프로젝트 II - 개발
- GitHub Copilot과 협업하여 프로젝트 구현
- 반복적 개선 및 테스트
- 동료 피드백

### 14주차: 디지털 리터러시에서 컴퓨팅 리터러시로
- 미래의 문해력과 필수 역량
- 프로젝트 발표 및 공유
- 상호 학습 및 피드백

### 15주차: 종합평가 및 미래 전망
- 최종 프로젝트 평가
- 누구나 창조자가 되는 시대
- 지속적인 학습을 위한 가이드
